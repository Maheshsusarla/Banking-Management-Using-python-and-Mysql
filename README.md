
# Bank Management System using Python and MySQL

A console-based Bank Management System developed using Python and MySQL. This project demonstrates database connectivity, CRUD operations, and modular programming concepts.

## Features

* Create Account
* Deposit Money
* Withdraw Money
* Check Balance
* View Customer Accounts
* Delete Account
* Menu-Driven Interface

## Technologies Used

* Python
* MySQL
* mysql-connector-python

## Concepts Applied

* Functions and Modules
* Database Connectivity
* CRUD Operations
* SQL Queries (INSERT, SELECT, UPDATE, DELETE)
* User Input Handling
